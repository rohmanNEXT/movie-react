## 🌟 Tentang Proyek

**Movie React** adalah aplikasi katalog film yang dibuat untuk keperluan pribadi dalam mengeksplorasi dan mengelola daftar film favorit. Berfokus pada performa dan antarmuka yang bersih untuk penggunaan sehari-hari.

## ✨ Fitur Utama

- 🎬 **Katalog Film:** Akses data film dengan detail lengkap.
- 🌈 **Tampilan Dark Mode:** Desain yang nyaman untuk penggunaan jangka panjang.
- ⚡ **Ringan & Cepat:** Navigasi instan antar halaman.
- 📱 **Responsif:** Dapat diakses dengan nyaman di perangkat seluler maupun desktop.
- 💾 **State Tracking:** Manajemen data lokal yang efisien.

## 🛠️ Teknologi

Aplikasi ini dibangun menggunakan:

| Komponen | Teknologi |
| :--- | :--- |
| **Library Utama** | React 19 / Vite |
| **Bahasa** | TypeScript |
| **Styling** | Tailwind CSS |
| **Manajemen State** | Redux Toolkit |
| **Lainnya** | Axios, Lucide React, Framer Motion |

## 🚀 Memulai

### 1. Instalasi
```bash
npm install
```

### 2. Jalankan
```bash
npm run dev
```

## 📸 Catatan Visual

Aplikasi ini menggunakan desain **Dark Mode** dengan transisi yang halus untuk memastikan pengalaman penggunaan pribadi yang optimal dan tidak melelahkan mata.
